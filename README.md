# Jav_Bus_Library

### 构成

练手项目

前端：Vue + element-ui + bootstrap-vue + nginx

后端：Gin + nginx

爬虫：Scrapy

数据库：Redis

### 启动

#### 网站服务器

```bash
$ python /JBL/docker_manage/manage.py 
$ 1
```

#### 爬虫

启动容器：

```bash
$ python /JBL/docker_manage/manage.py 
$ 2
```

在容器内运行爬虫：

```bash
$ python main.py 
```
